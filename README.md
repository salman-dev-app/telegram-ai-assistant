## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Scroll.png" alt="About" width="30" height="30" style="vertical-align:middle;" /> About This Project

A production-grade Telegram AI Assistant built with Node.js/JavaScript. It supports multiple AI model backends, persistent conversation history via MongoDB, human-like group interaction, smart topic detection, and dynamic response personalities.
