### Prerequisites

Make sure you have the required tools installed before running this project.

### Installation

```bash
# Clone the repository
git clone https://github.com/salman-dev-app/telegram-ai-assistant.git
cd telegram-ai-assistant

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env
# Add BOT_TOKEN, GROQ_API_KEY, MONGODB_URI

# Run locally
npm start

# Or deploy to Koyeb using the button in the README
```
